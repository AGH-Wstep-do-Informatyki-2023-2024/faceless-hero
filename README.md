# Faceless Hero

## Wstęp

Dzięki tym instrukcjom otrzymasz kopię projektu na twoim komputerze lokalnym i będziesz mógł uruchomić grę.

#### Wymagania wstępne

```
pip install pygame
```

#### Instalacja

```
git clone https://github.com/AGH-Wstep-do-Informatyki-2023-2024/faceless-hero.git
```

#### Rozpoczęcie gry

```
python main.py
```
